# Lab 03: NLP, Vision, and Search Integration

## Objective

Map mixed requirements to NLP, vision, and search capabilities.

## Scenario

A logistics company wants to:
- Extract text from package labels (images)
- Detect complaint sentiment from customer notes
- Search historical incidents and generate response suggestions

## Your Task

1. Propose service mapping for each requirement.
2. Draw a basic data flow.
3. Identify two likely failure points.
4. Propose one monitoring metric per component.

## Grading Rubric (5 points)

- 1 point: Correct capability mapping
- 1 point: End-to-end flow clarity
- 1 point: Failure analysis depth
- 1 point: Monitoring metric relevance
- 1 point: Practical mitigation proposals
