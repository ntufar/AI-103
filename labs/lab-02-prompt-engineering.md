# Lab 02: Prompt Engineering

## Objective

Design and improve prompts for reliability, structure, and safety.

## Tasks

1. Write a baseline prompt for: "Summarize customer feedback and propose top 3 actions."
2. Improve it using explicit role, constraints, and output schema.
3. Add safety constraints for harmful or missing-context requests.
4. Evaluate output with a 5-point rubric.

## Scoring Rubric (5 points)

- 1 point: Prompt structure completeness
- 1 point: Output format control
- 1 point: Groundedness strategy
- 1 point: Safety/guardrails
- 1 point: Iteration quality (before vs after)

## Output Template

- Baseline prompt
- Improved prompt
- Example output
- Quality review notes
- Final score
