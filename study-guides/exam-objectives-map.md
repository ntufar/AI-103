# AI-103 Exam Objectives Map

Use this map to align your practice with core AI-103 skill areas.

## 1) Plan and manage an Azure AI solution

Focus areas:
- Select the right Azure AI service for a business requirement
- Understand pricing tiers, regions, quotas, and service limits
- Design for security and responsible AI principles
- Plan monitoring, logging, and lifecycle management

Practice outcomes:
- You can justify why to use Language, Vision, Speech, Search, or OpenAI
- You can propose architecture trade-offs in terms of cost and scale

## 2) Implement generative AI solutions

Focus areas:
- Prompt engineering basics and iterative prompt refinement
- Grounding responses with enterprise data (RAG pattern)
- Safety filters, content moderation, and responsible responses
- Chat completions, system prompts, and evaluation loops

Practice outcomes:
- You can design prompts with role, task, constraints, and examples
- You can evaluate output quality and reduce hallucinations

## 3) Implement natural language processing solutions

Focus areas:
- Sentiment and opinion mining
- Named entity recognition and key phrase extraction
- Summarization and language detection
- Conversational language understanding concepts

Practice outcomes:
- You can choose the right NLP capability per requirement
- You can interpret confidence scores and failure modes

## 4) Implement computer vision solutions

Focus areas:
- Image analysis and object/tag detection concepts
- OCR (read text from images) use cases and constraints
- Content understanding and image metadata interpretation

Practice outcomes:
- You can map image tasks to the correct service
- You can reason about quality, preprocessing, and expected errors

## 5) Implement knowledge mining and document intelligence

Focus areas:
- Search indexes, vector fields, and chunking strategy
- Skillsets, indexers, and enrichment pipeline concepts
- Retrieval quality, relevance tuning, semantic ranking

Practice outcomes:
- You can design a high-level indexing and retrieval pipeline
- You can explain how chunking affects answer quality

## Weekly Domain Coverage Checklist

- [ ] Plan and architecture decisions
- [ ] Generative AI and prompt engineering
- [ ] NLP tasks and interpretation
- [ ] Vision and OCR capabilities
- [ ] Search/RAG/knowledge mining design
- [ ] Responsible AI and risk mitigation
