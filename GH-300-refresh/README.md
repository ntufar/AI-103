# GH-300: GitHub Copilot Certificate - Exam Prep

**Exam Date:** April 17, 2026  
**Exam Duration:** 100 minutes  
**Passing Score:** 700+  
**Exam Format:** Proctored assessment with interactive components

## Exam Overview

The GitHub Copilot Certificate (GH-300) tests your expertise in using GitHub Copilot to improve software development productivity, quality, and security. The exam covers responsible AI use, prompt engineering, Copilot features, and privacy safeguards.

## Key Facts Before You Start

- ✅ **100 questions in 100 minutes** - Pace yourself (~1 min per question)
- ✅ **Score needed: 700+** - This is typically ~70% correct
- ✅ **Hands-on scenarios** included - May have interactive coding challenges
- ✅ **Focus on practical usage** - Exam tests real-world application, not theory alone
- ✅ **Feature updates** - Some preview features may appear; general availability (GA) features are primary focus

## 7 Skill Areas Tested

| Skill Area | Weight | Focus |
|-----------|--------|-------|
| **Responsible AI & Ethical Use** | 15-20% | Risks, limitations, bias mitigation, responsible operation |
| **Copilot Features (IDE & CLI)** | 25-30% | Inline suggestions, chat, Plan Mode, CLI commands, organizations settings |
| **Advanced Features** | 25-30% | Agent Mode, Edit Mode, MCP, Code Review, Spaces, Spark, Pull Requests |
| **Data & Architecture** | 10-15% | Data handling, input processing, code suggestion lifecycle, LLM limitations |
| **Prompt Engineering** | 10-15% | Prompt structure, context, zero/few-shot, best practices |
| **Developer Productivity** | 10-15% | Code generation, refactoring, testing, security improvements |
| **Privacy & Safeguards** | 10-15% | Content exclusions, duplication detection, security warnings |

### Exam Day (April 17)
- Get good sleep tonight

- Flag uncertain answers for review if time permits
- Don't second-guess correct answers

## Top 10 Topics to Know Cold

1. **Copilot Chat** - Limits, commands, prompt reuse, instructions files
2. **Agent Mode & Sub-Agents** - Context optimization, task delegation
3. **Prompt Structure** - How to craft effective prompts for different scenarios
4. **Data Privacy** - What data Copilot sees, how it's used, retention policies
5. **Content Exclusions** - How to exclude sensitive files/repos
6. **Duplication Detection** - How Copilot handles potentially duplicated code
7. **Responsible AI** - Biases, limitations, validation needs
8. **Copilot CLI** - Installation, key commands, file management
9. **Code Review Features** - Policies, audit logs, standards management
10. **Plan Mode vs Edit Mode** - When to use each, differences
