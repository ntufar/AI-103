# Responsible AI Checklist

## Before Deployment

- [ ] Define intended use and prohibited use
- [ ] Identify impacted users and possible harms
- [ ] Add abuse-case and edge-case tests
- [ ] Define fallback behavior for uncertain outputs

## During Development

- [ ] Add input validation and output checks
- [ ] Add prompt-level safety instructions
- [ ] Add content moderation where applicable
- [ ] Track model/version changes and impacts

## Monitoring

- [ ] Log safety incidents and resolution path
- [ ] Monitor drift in response quality
- [ ] Track user feedback and false positive/negative rates
- [ ] Periodically review and tighten guardrails

## Exam Tip

For architecture questions, mention:
- Fairness
- Reliability and safety
- Privacy and security
- Inclusiveness
- Transparency
- Accountability
